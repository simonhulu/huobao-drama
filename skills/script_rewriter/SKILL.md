# 剧本改写指南

## 改写原则

1. **保留核心情节**：不改变主线故事和角色关系
2. **增强画面感**：将叙述性文字转化为可视化的场景描写
3. **对白逐字保留**：原文中已有的角色台词必须逐字照搬进剧本，不得改写、润色、精简或替换台词本身；原文没有的对白不许凭空编造；rewriter 只负责场景切分、角色标注和补充画面感的动作描写
4. **节奏把控**：每场戏控制在 30-60 秒，适合短视频
5. **不写镜头语言**：不涉及景别、角度、运镜，这些属于分镜拆解步骤

## AI 漫剧改写模式

当任务明确要求“AI 漫剧”“竖屏短剧”“快节奏”或输入内容来自 `ai_manga_drama` 风格的分集时，启用以下附加规则：

1. **只保留情节需要和情绪完整的内容**
   - 删除大段背景说明、重复的心理描写、与主线无关的回忆。
   - 合并情绪功能重复的场次，只保留情绪落差最大的那一次。

2. **对白可压缩，不必逐字保留**
   - 允许合并、删减口语冗余，只保留推进冲突、承载情绪、制造反转的台词。
   - 原文没有的对白仍然不许编造。

3. **每场戏必须有强冲突或情绪转折**
   - 场景之间用情绪推进：压抑 → 爆发 → 反杀 → 释然。
   - 不写纯过渡场景。

4. **输出增加情绪标注**
   - 在场景头或动作描写中用简短标注提示情绪走向，例如 `【压抑】`、`【爆发】`、`【反杀】`、`【释然】`。

5. **节奏更紧**
   - 单场戏控制在 20-45 秒，整集对白和动作高度压缩。

## 格式化剧本格式

```
## S01 | 内景 · 咖啡厅 | 黄昏

黄昏的光线透过落地窗洒进咖啡厅，吧台上咖啡杯热气升腾。

小明独自坐在角落卡座，低头看手机，神情有些焦虑。

门铃响起，小红推门而入。她看到小明，微笑着走过去。

小红：（微笑）等很久了吗？
小明：（抬头）还好，刚到。
```

### 格式规则

- `## S编号 | 内景/外景 · 地点 | 时间段` — 场景头
- 动作描写自然段 — 不包含任何镜头语言
- `角色名：（状态/表情）台词内容` — 对白格式

### 内容量参考

格式化剧本相比原始内容增加约 20-30%，主要增量是场景头标记和对白格式化，不是扩写。

## 改写步骤

1. 先调用 `read_episode_script` 读取原始内容
2. 分析内容结构（对话、叙述、心理描写的比例）
3. 调用 `rewrite_to_screenplay` 执行改写
4. 检查改写结果，确认符合格式化剧本格式
5. 调用 `save_script` 保存最终结果

## 注意事项

- 对白逐字保真是硬要求：可以调整对白的归属和场景位置，但台词文字本身保持与原文一致
- 心理描写可转化为角色表情/动作或画外音
- 长段叙述拆分为多个短场景
- 确保每个场景有明确的情绪转折点
- 保持角色语言风格一致性
- 场景编号连续递增（S01, S02, S03...）
- 时间段要具体（黄昏、深夜、清晨），不要笼统写"白天"
